# Space Fleet (Sinclair BASIC)

This is a Sinclair BASIC type-in for the 48k ZX Spectrum, based on the 90s board game Space Fleet.

.tap file for use with a ZX Spectrum emulator (tested with Fuse).

[Download .tap file](https://github.com/adambient/spacefleet/releases/download/v1.0/space-fleet.tap)
