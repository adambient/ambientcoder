# ZX Life 2.0

This is version 2.0 of an implementation of Conway's Game of Life for the ZX Spectrum 16k and above. 128k or AY interface required to play the music (Dragon's Lair 2 - Subtune 2 (Rapids) by Rob Hubbard).

.tap file for use with a ZX Spectrum emulator (tested with Fuse).

[Download .tap file](https://github.com/adambient/zx-life/releases/download/v2.0/zx-life.tap)
