# Blitz (Sinclair BASIC)

This is my first attempt at using some "assembly assist" in Sinclair BASIC. That is, loading a machine code routine using DATA statements to replace the PRINT statement with something quicker and easier to use for game graphics in Sinclair BASIC.

.tap file for use with a ZX Spectrum emulator (tested with Fuse).
